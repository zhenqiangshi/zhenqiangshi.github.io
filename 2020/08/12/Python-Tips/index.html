Python Tips

一、signal

Python中对信号进行处理的模块主要使用signal模块。

用途：接受一个进程关闭的信号，然后收到信号后，处理收尾工作，然后再关闭此进程。比如实时处理消息队列的数据，发生异常崩溃后，可以将内存临时数据及时的保存为文件或者入库。

二、zmq

使用其一对多的发布订阅模式，可以用于过滤数据。也可以在消费数据时进行数据分发，分发到多个进程进行处理，提高处理性能。
